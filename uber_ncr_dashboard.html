MySQL
Raw dataset had 150,000 rows. Used DELETE FROM rides WHERE id NOT IN (SELECT MIN(id) FROM rides GROUP BY booking_id) to remove 1,233 duplicate Booking IDs. Final clean table: 148,767 rows × 21 columns. All aggregations run via GROUP BY, WINDOW FUNCTIONS, and CASE WHEN.
